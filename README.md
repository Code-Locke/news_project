# news_project
A news aggregator that gives personal geopolitical intelligence and news on a dashboard format.

At the current form, this python file reads from the txt file, and use the feedparser library to read the RSS feeds and show them in the terminal. This will change to store the feeds and stories in the backend SQLite

The current frontend flask app is rudimentary, and will eventually use Tailwind or other CSS styling templates to clean them up

This project is done with the intention to be ran on a linux environment with cron jobs to automate things.
